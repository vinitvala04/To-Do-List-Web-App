# 🚀 Responsive To-Do List Web App

A modern, fully responsive To-Do List web application built using **HTML, CSS, and JavaScript**.  
This app helps users manage daily tasks with a clean interface, dark mode, and persistent storage.

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📊 Progress counter (Completed / Total)
- 📅 Current date display
- 🌙 Dark mode toggle
- 💾 Auto-save using Browser LocalStorage
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- ⚡ Fast and lightweight (No libraries required)

---

## 📸 Preview

> A clean task manager that works offline and stores data locally in your browser.

---

## 🛠️ Technologies Used

- **HTML5** — Structure
- **CSS3** — Styling & Responsive Design
- **JavaScript (ES6)** — Functionality & Storage
- **LocalStorage API** — Data persistence

## 💾 Data Storage

This app uses **Browser LocalStorage**.

✔ Tasks are saved automatically  
✔ Works offline  
✔ No account required  

⚠️ Data is stored only on your device and browser.

Data may be lost if:

- Browser data is cleared
- Incognito mode is used
- Device is changed
